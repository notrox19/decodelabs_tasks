# 💰 Expense Tracker

A full-stack application to manage daily income and expenses.

## 🚀 Features
- Add, edit, delete transactions
- Track income & expenses
- Live balance calculation
- Filter by category

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot
- Database: MySQL

## ▶️ How to Run
1. Start backend (Spring Boot)
2. Open index.html
3. Manage your expenses
